Introduction to Computer Networks

A computer network is a set of computers sharing resources located on or provided by network nodes.
The Internet is the largest computer network in the world and refers to a global system of interconnected networks.

Protocols

A protocol is a set of rules that defines how data is transmitted between devices in a network.
TCP/IP is the foundational protocol suite of the Internet. TCP stands for Transmission Control Protocol.
IP means Internet Protocol and is responsible for addressing and routing packets.
HTTP is a protocol that defines how web browsers and servers communicate.
DNS means Domain Name System and refers to a service that translates domain names into IP addresses.

OSI Model

The OSI model is a conceptual framework that standardizes the functions of a communication system into seven layers.
The physical layer is the lowest layer and refers to the transmission of raw bits over a physical medium.
The data link layer is responsible for node-to-node data transfer and error detection.
The network layer is responsible for packet forwarding and routing through intermediate routers.
The transport layer provides end-to-end communication services for applications.
The session layer manages sessions between applications.
The presentation layer is responsible for data translation, encryption, and compression.
The application layer is the topmost layer and refers to the network services provided to applications.

IP Addressing

An IP address is a numerical label assigned to each device connected to a computer network.
IPv4 uses 32-bit addresses and supports about 4.3 billion unique addresses.
IPv6 is the newer version of IP that uses 128-bit addresses, providing a vastly larger address space.
A subnet mask is used to divide an IP address into network and host portions.
NAT means Network Address Translation and refers to a method of mapping private IP addresses to a public IP address.

Routing

A router is a networking device that forwards data packets between computer networks.
Routing is the process of selecting a path for traffic in a network.
BGP is Border Gateway Protocol and is used for routing between autonomous systems on the Internet.
OSPF means Open Shortest Path First and is a link-state routing protocol used within a single autonomous system.

Wireless Networks

WiFi is a technology that allows devices to connect to a local area network wirelessly.
Bluetooth is a short-range wireless technology used for connecting devices over short distances.
LTE means Long-Term Evolution and refers to a standard for wireless broadband communication for mobile devices.
A MAC address is a unique identifier assigned to network interface controllers for communications at the data link layer.
